# LOKASAMYOGA - Evaluation 2 Report

## 1. Project Overview

### Project Title
**LOKASAMYOGA**

### Project Goal
The goal of Lokasamyoga is to create a centralized digital platform that connects NGOs, Volunteers, Donors, and Admins to collaborate efficiently. The system simplifies event management, volunteer participation, and donation communication while ensuring transparency and structured administration.

---

## 2. Project Features (Final)

### 2.1 User Roles
The system accommodates the following distinct user roles, each with specific permissions and features:
1. **NGO**
2. **Volunteer**
3. **Donor**
4. **Admin** (Basic)

### 2.2 NGO Features
* **Registration & Login**: Secure sign-up and authentication for NGOs.
* **Event Management**: Create social events providing essential details such as:
    * Event Title
    * Event Date
    * Event Location
    * Event Description
* **Volunteer Management**: View a comprehensive list of volunteers registered for their specific events.

### 2.3 Volunteer Features
* **Registration & Login**: Secure sign-up and authentication for volunteers.
* **NGO Discovery**: View a curated list of NGOs categorized or filtered by location.
* **Event Discovery**: View upcoming and completed events.
* **Event Participation**: Register for specific events of interest.
* **NGO Association**: Join and follow specific NGOs to stay updated on their activities.

### 2.4 Donor Features
* **Registration & Login**: Secure sign-up and authentication for donors.
* **NGO Discovery**: View the list of registered and approved NGOs.
* **Event Discovery**: View events created by NGOs to understand their active initiatives.
* **Donation Communication**: Directly contact NGOs for donation purposes. *(Note: The current scope does not include an online payment gateway).*

### 2.5 Event Features
* **Creation & Ownership**: Events are exclusively created and managed by NGOs.
* **Visibility**: All events are visible to both Volunteers and Donors.
* **Status Tracking**: Events are tagged with clear statuses such as **Upcoming** or **Completed**.
* **Volunteer Registration**: Seamless integration for volunteers to register for events directly from the event details page.

### 2.6 Admin Features
* **NGO Oversight**: View a comprehensive list of all registered NGOs.
* **NGO Approval Workflow**: Authority to Approve or Reject new NGO registrations to maintain platform integrity and transparency.
* **User Management**: Read-only access to view all users (NGOs, Volunteers, Donors) registered on the platform.

---

## 3. Future Scope & Enhancements
* Integration of an online payment gateway for secure, direct donations.
* Advanced analytics and reporting for Admins and NGOs.
* In-app messaging and notification systems for real-time updates.
